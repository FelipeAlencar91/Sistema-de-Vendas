# Sistema de Análise de Vendas com Python e Excel

Este projeto realiza a análise de vendas a partir de um arquivo Excel, utilizando Python e a biblioteca Pandas.

O sistema calcula automaticamente:

- Faturamento total
- Produto mais vendido
- Produto menos vendido
- Faturamento por produto
- Exportação de relatório em Excel

## Como executar o projeto

1) Instalar dependências:

pip install pandas openpyxl

2) Executar o sistema:

python src/sistema_de_vendas.py

---

## Funcionalidades

O sistema:

- Lê dados de vendas de um arquivo Excel
- Calcula o total de cada venda
- Calcula o faturamento total
- Identifica o produto mais vendido
- Identifica o produto menos vendido
- Calcula o faturamento por produto
- Exporta um relatório em Excel automaticamente
